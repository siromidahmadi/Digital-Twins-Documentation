Goal of the Project

Build a premium, vector-enabled Earth visualization assistant — Digital Twins Project — that integrates real-time datasets, AI-guided navigation and intuitive explorations of environmental features.

  • Live data feeds (wildfires, earthquakes, weather)
  • 3D navigation & vector paths
  • Voice assistant & contextual guidance
