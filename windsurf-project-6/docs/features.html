Key Features

Real-time Data Integration

Aggregates USGS, NASA FIRMS, OpenWeatherMap, OpenAQ and other feeds into a unified GeoJSON pipeline with configurable refresh intervals.

Vector Navigation & 3D Routes

Create and visualize vector routes, measure distances, and render smooth animated camera transitions over the globe.

AI Assistant & Contextual Guidance

Natural-language assistant that answers questions about locations, suggests points of interest, and explains dataset anomalies.

Problem It Solves

Complex, multi-source geospatial data is often scattered and hard to navigate. Digital Twins Project centralizes feeds, standardizes formats and puts intuitive navigation and interpretive AI on top so analysts and decision-makers can act faster.

Data Type It Works With

  • GeoJSON - features & geometries
  • CSV/TSV - imported sensor logs
  • Raster tiles - satellite imagery / basemaps
  • Time series - sensor readings, telemetry
  • Binary - model outputs, optimized vector tiles
